> **目标**：查漏补缺，覆盖 Easy / Medium，不碰 Hard。
> **节奏**：每天 30 分钟，每周 5 天学习 + 2 天复盘。
> **方法**：按专题滚动复习，先复习已有代码，再补做同类型新题。
> **整理原则**：每天练习过程中，顺手把经验归类整理到对应目录的 README.md。

## 四、核心模板速查（必须能默写）

### 1. 快速排序（分区函数）

```java
public int partition(int[] nums, int start, int end) {
    int pivot = nums[end];
    int i = start;
    for (int j = start; j < end; j++) {
        if (nums[j] <= pivot) {
            swap(nums, i++, j);
        }
    }
    swap(nums, i, end);
    return i;
}
```

### 2. 二分查找（左闭右闭）

```java
public int binarySearch(int[] nums, int target) {
    int left = 0, right = nums.length - 1;
    while (left <= right) {
        int mid = left + (right - left) / 2;
        if (nums[mid] == target) return mid;
        else if (nums[mid] < target) left = mid + 1;
        else right = mid - 1;
    }
    return -1;
}
```

### 3. 链表反转（迭代）

```java
public ListNode reverseList(ListNode head) {
    ListNode pre = null, cur = head;
    while (cur != null) {
        ListNode next = cur.next;
        cur.next = pre;
        pre = cur;
        cur = next;
    }
    return pre;
}
```

### 4. 回溯模板

```java
void backtrack(路径, 选择列表) {
    if (满足结束条件) {
        result.add(路径);
        return;
    }
    for (选择 : 选择列表) {
        做选择;
        backtrack(路径, 选择列表);
        撤销选择;
    }
}
```

### 5. BFS 模板

```java
void bfs(Node start) {
    Queue<Node> queue = new LinkedList<>();
    Set<Node> visited = new HashSet<>();
    queue.offer(start);
    visited.add(start);
    while (!queue.isEmpty()) {
        Node cur = queue.poll();
        for (Node neighbor : cur.neighbors) {
            if (!visited.contains(neighbor)) {
                visited.add(neighbor);
                queue.offer(neighbor);
            }
        }
    }
}
```

### 6. DFS 模板（岛屿问题）

```java
void dfs(char[][] grid, int i, int j) {
    if (i < 0 || i >= grid.length || j < 0 || j >= grid[0].length || grid[i][j] == '0')
        return;
    grid[i][j] = '0'; // 淹没
    dfs(grid, i + 1, j);
    dfs(grid, i - 1, j);
    dfs(grid, i, j + 1);
    dfs(grid, i, j - 1);
}
```

int maxEndingHere = nums[0], maxSoFar = nums[0];
for (int i = 1; i < nums.length; i++) {
    maxEndingHere = Math.max(nums[i], maxEndingHere + nums[i]);

### 8. 并查集模板

```java
class UnionFind {
    int[] parent, rank;
    int count;
    UnionFind(int n) {
        parent = new int[n];
        rank = new int[n];
        for (int i = 0; i < n; i++) parent[i] = i;
        count = n;
    }
    int find(int x) {
        if (parent[x] != x) parent[x] = find(parent[x]);
        return parent[x];
    }
    void union(int x, int y) {
        int px = find(x), py = find(y);
        if (px == py) return;
        if (rank[px] > rank[py]) parent[py] = px;
        else if (rank[px] < rank[py]) parent[px] = py;
        else { parent[px] = py; rank[py]++; }
        count--;
    }
}
```

## 五、执行建议

1. **每天固定时间**：建议早晨或睡前 30 分钟，形成肌肉记忆。
2. **先复习后新题**：确保已有代码能默写，再拓展新题。
3. **标记难度**：每道题标记 🟢（秒写）/ 🟡（需思考）/ 🔴（需看答案），滚动攻克 🔴。
4. **周末复盘**：重做错题，整理「易错点清单」，顺手更新对应目录的 README.md。
5. **12 周循环**：完成一轮后，从 Week 1 重新开始，你会发现 🟢 越来越多。
6. **边练边整理**：每天最后 5 分钟，把当天的心得、易错点、新题解整理到对应目录的 README.md，12 周后自然形成完整的知识体系。
