#三 回答问题和按要求写出函数定义
1.递归函数计算T(n)

```
int T(int n){
    if(n==0)return 0;
    else if(n<=2)return 1;
    else
        return T(n-1)+T(n-2)+T(n-3);
}
```
2.非递归函数计算T(n)

```
int T(int n){
    int i,t0=0,t1=1,t2=1,s=0;
    if(n==0)return 0;
    else if(n<=2)return 1;
    for(i=3;i<=n;i++){
        s=t0+t1+t2;
        t0=t1;
        t1=t2;
        t2=s;
    }
    return s;
}
```
